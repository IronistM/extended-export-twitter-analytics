Extended Export for Twitter Analytics

  1. Log into the Twitter account you need data for. (You can do this in a separate tab or window, but not a separate browser.)
  2. Go to https://ads.twitter.com/accounts/xxxxxx/timeline_activity/tweet_data (opens in new tab)
  3. You should see a page with a lot of text and curly brackets, starting with the word "statuses". Paste that into this box:
  4. If the data you pasted is valid, a table should appear below and the Download as CSV link should light up. Yay!
  5. Click to download as CSV.
