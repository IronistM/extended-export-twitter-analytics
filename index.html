Extended Export

for Twitter Analytics

  1. Log into the Twitter account you need data for. You can do this in a separate tab or window, but not a separate browser.
  2. Click here to go to https://ads.twitter.com/accounts/xxxxxx/timeline_activity/tweet_data opens in new tab
  3. You should see a page with a lot of text and curly brackets, starting with the word "statuses". Copy & paste all of the text into this box:

  4. Select a date range: Optional. Blank defaults to All

    • All
    • 30 Days
    • 60 Days
    • 90 Days

  5. Select the columns you want:

    Date & Time

    • Unix Timestamp Time in milliseconds; most precise for sorting. (What's Unix time?)
    • ISO Timestamp Format: 2014-04-22T17:55:05.000Z (What's ISO time?)
    • ISO Date Format: 2014-04-22
    • MM/DD/YYYY

    Basic Info

    • Tweet Text
    • Permalink
    • Tweet ID

    Stats

    • Retweets
    • Faves
    • Replies
    • Reach Value is a multiplier (i.e. 2.99 x normal reach)

    Sponsored & Campaign Info

    • Is Sponsored? Boolean
    • Is Nullcasted? Boolean
    • Is Narrowcasted? Boolean
    • Campaign Info Returns names of campaigns this tweet was on.

    Links

    • Mentions
    • Links with Click Counts Includes embedded photo
    • Hashtags
  6. Download as CSV
