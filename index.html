Extended Export

for Twitter Analytics

  1. Log into the Twitter account you need data for. You can do this in a separate tab or window, but not a separate browser.
  2. Click here to go to https://ads.twitter.com/accounts/xxxxxx/timeline_activity/tweet_data opens in new tab
  3. You should see a page with a lot of text and curly brackets, starting with the word "statuses". Copy & paste all of the text into this box:

  4. Select the properties you want:

    Basic Info

    • Date & Time Required
    • Tweet Text
    • Permalink
    • Tweet ID

    Stats

    • Retweets
    • Faves
    • Replies

    Sponsored Info Each returns "true" or "false" and also returns sponsored start & end dates, campaign names, and campaign start & end dates

    • Is Sponsored?
    • Is Nullcasted?
    • Is Narrowcasted?

    Links

    • Mentions
    • Links with Click Counts Includes embedded photo links
    • Hashtags

    Huh?

    • Tweet Index
    • Badges Currently does not work until I figure out what this is
  6. If the data you pasted is valid, a table should appear below and the Download as CSV link should light up. Yay!
  7. Click to download as CSV.
  • As of now, "clicks" only displays the click count for the first link in the tweet (whether it's an uploaded photo or pasted-in link). Still working on getting clicks for each link.
